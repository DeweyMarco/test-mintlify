<Update label="2026-04-02" description="April 2026" tags={["New", "Removed"]}>
  ## New "marco original" category

  A new `marco original` category is now available with an original quote by Marco Dewey. You can filter by this category using the [Get random quote](/api-reference/get-random-quote) or [List quotes](/api-reference/list-quotes) endpoints.

  ## Humor category removed

  The `humor` category and all of its quotes have been removed from the API. Requests filtering by `humor` will now return an error. Update any existing integrations that reference this category. See the [List categories](/api-reference/list-categories) endpoint for active categories.
</Update>
