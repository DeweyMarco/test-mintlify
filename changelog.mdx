<Update label="2026-04-02" description="April 2026" tags={["New", "Deprecation"]}>
  ## New "marco original" category

  A new `marco original` category is now available with an original quote by Marco Dewey. You can filter by this category using the [Get random quote](/api-reference/get-random-quote) or [List quotes](/api-reference/list-quotes) endpoints.

  ## Humor category deprecated

  The `humor` category has been deprecated and all of its quotes have been removed. The category still appears in the API schema for backward compatibility, but it no longer returns results. Avoid using it in new integrations—it will be fully removed in a future release. See the [List categories](/api-reference/list-categories) endpoint for active categories.
</Update>
