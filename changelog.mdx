<Update label="2026-04-02" description="April 2026" tags={["New", "Update"]}>
  ## New category and quote

  A new `marco original` category is now available, featuring an original quote by Marco Dewey. You can filter by this category using the [List quotes](/api-reference/list-quotes) or [Get random quote](/api-reference/get-random-quote) endpoints.

  ## Humor category removed

  The `humor` category and its associated quotes have been removed from the API. Requests filtering by `humor` will now return an error. See the [List categories](/api-reference/list-categories) endpoint for current valid categories.
</Update>
