## Overview

The Quotes API is a simple REST API that serves a curated collection of quotes across categories including **motivation**, **wisdom**, **philosophy**, **technology**, and **marco original**.

It's designed to be easy to integrate — no authentication required, JSON responses, and predictable error shapes. The API ships with both Node.js and Python server implementations.

## Base URL

```
http://localhost:3000
```

## Available Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/quotes` | List all quotes, with optional filters |
| `GET` | `/quotes/random` | Get a random quote |
| `GET` | `/quotes/{id}` | Get a specific quote by ID |
| `GET` | `/categories` | List available categories |

## Response Format

All successful responses return JSON. Errors follow a consistent shape:

```json
{
  "error": "invalid_category",
  "message": "Category 'foo' does not exist. Valid categories: marco original, motivation, philosophy, technology, wisdom"
}
```

## Categories

You can filter quotes by category. The active categories are:

| Category | Description |
|----------|-------------|
| `marco original` | Original quotes by Marco |
| `motivation` | Motivational and inspirational quotes |
| `philosophy` | Philosophical quotes |
| `technology` | Technology-related quotes |
| `wisdom` | Quotes about wisdom and life lessons |

<Warning>
  The `humor` category has been removed. Requests that filter by `humor` now return an error. Avoid using it in existing integrations.
</Warning>

## No authentication required

This API is open — no API keys or tokens needed.
